关于云计算的话题越来越多，我们常常如坠云雾不知所云。如果你不懂什么是云计算，或者看不到云计算存在的理由，说明你声处一个好公司。Gartner 的分析师认为，在很多场合下，云计算不过是新瓶装旧酒。
微软 CEO Steve Ballmer 最近在 Churchill Club 的一次露面中说，当人们谈到云计算，他指的是将一些东西拿到防火墙的外面，可能是放到一个共享服务器，或是存储系统中去。
Oracle CEO Larry Ellison 说得更绝，根据 Wall Street Journal 的报道，Ellison 在最近的一次分析师大会上说，计算机业是唯一比女人时装更爱追求时髦的行业。或许我是个白痴，我不知道人们说的云计算是什么东西。完全是胡话，简直疯了，这种傻事到底什么时候才是头？
据 Gartner 分析公司的说法，这些混乱来自那些硬件和软件厂商。Gartner 的副总裁兼分析师 David Smith 说，他认为云计算是新瓶装旧酒，是厂商们期待提升他们的产品销售时抛出的一个时髦词。
Ballmer 和 Ellison 并非唯一抨击这个潮流的人，随着对云计算持续增长的失望，自由软件基金会干将 Richard Stallman 也加入批评的行列，在接受 UK Guardian 采访时，他特别对 Google 的 Gmail 提出批评，他说，这是一个圈套，强迫人们购买一个封闭的，专用的系统，最终人们的花费将越来越大。很愚蠢，比愚蠢还坏，他说，这是一种市场炒做。
该责怪厂商吗?
Gartner 的 Smith 说，他明白 Ballmer 和 Ellison 批评云计算是出于对和自己存在竞争关系的软件模式的一种抗拒，但他们也是对这种因滥用概念而带来的混乱的一种反应。
事实上，云计算在各种产品和服务中已广泛存在。Sell 曾企图注册云计算商标，用于其服务器产品，却被专利商标局拒绝。
而微软与 Oracle 也并没真正对云计算敬而远之。正在参加云计算展的 Ballmer 宣布微软将开发一种新的 Windows 平台，Windows Cloud，用于支撑基于云的应用。IDG 透露，Windows Cloud 的细节将随本月月底的 PDC 公布。
与此同时，Amazon 公开了他们最新的云服务，允许用户在他们的 EC2 (Elastic Coumpute Cloud) 服务上运行 Windows 服务器与 SQL Server 数据库。上周，Amazon 和 Oracle 联合演示可以运行在 EC2 上的 Oracle 软件。Smith 说，这些开发工作正是部分问题所在。一方面，人们有一种冲动，将所有的东西都冠以云的名号，因为这很时髦。厂商需要这些来促进销售，IT 部门需要在项目中加入云的概念以获得预算。
将“云模式技术”和“云计算服务”混淆也带来更多混乱，Smith 补充道。Dell 的服务器和 Salesforce.com CRM 系统如何可以都叫云计算，它们根本不是一回事。相反，Smith 建议将它们分为云计算服务和云模式技术。或者使用以前已经有的名称，硬件和软件。
更重要的是，用户应该忘记这些时髦词，问问自己，我要的是什么？对客户端/服务器技术，电子商务以及基于服务的架构，Smith 的建议也是一样的。这些都是新瓶装旧酒而已。
