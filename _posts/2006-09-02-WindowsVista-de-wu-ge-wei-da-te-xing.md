5. Windows ReadyBoost
首次听说这个功能像是科幻小说一般：Windows Vista 的 ReadyBoost 功能可以使用USB 2.0闪存加速Vista PC的性能。在一阵傻笑之后，我意识到这项功能不是玩笑，它是认真的。

当然，有个先决条件，这就是USB存储器必须满足一定的性能和容量要求：2.5MB/s传输、4K的随机读取速度，1.75MB/s的传输、512K的随机写入速度；64MB到8GB的空闲空间；总容量256MB以上。

ReadyBoost 把 USB 存储器的空间当作系统内存使用，以此运行。在首次插入兼容设备时，Vista的自动播放对话框将会出现"Speed up my system"选项（附图）。然后就可以在该设备的属性对话框中决定给 ReadyBoost 分配多少使用容量（附图）。需要注意的是，这部分容量就不能够使用了，除非格式化设备或关闭/改变ReadyBoost容量。你不能够在多台Vista机器上只使用一个USB闪存，也不能在一台机器上使用多个USB闪存。并且，Vista将会建议设置空间的大小。

这个功能为什么好呢？对于一般的桌面PC，添加RAM并不困难（要有钱才行啊！），而对于多数笔记本而言，可能受制于各种条件和要求则需要扩充系统RAM（内存）。不管怎么样，你只需要插入USB闪存、稍微配置即可提升系统性能。

有一些担心说 ReadyBoost 技术会缩短闪存的寿命，因为闪存有一定数量的读取和写入次数限制。Paul 的看法是，谁在乎呢？这些东西都很便宜了，如果你聪明的话，可以为此准备一个专用的USB闪存。如果它完蛋了就再换一个。（除此以外，微软也指出，研究表明使用 ReadyBoost 可以让闪存获得10年或更多的使用寿命。）

ReadyBoost 将给小内存 PC 最大的性能提升，当然与闪存的容量有关。1GB 的 USB 闪存与 512MB 的 USB 闪存差异是巨大的，对于拥有 4GB 内存的PC，添加闪存没有什么意义。

4. 整合的搜索

当微软首次在 2003 年宣布基于整合桌面搜索的操作系统合法化时，这听起来是个很好的注意，以致在2005年苹果将Spotlight 作为 Mac OS X 10.4 "Tiger"的一部分推向市场，这早于 Vista 一年半时间。而且，Google 也开始了桌面搜索产品在微软行动迟缓的操作系统上和巨大的 XP 用户群中获利。

但是，你知道吗？Vista 的搜索技术要比其竞争对手的要好得多。许多人错误的人为由于从 Vista 中删除了 WinFS 存储引擎将会导致这个部分的倒退。相反，Vista 实际上让终端用户可以从 WinFS 上获取的这部分的所有功能，虽然使用的是与 Spotlight 和 MSN Desktop Search 同样的更老的索引搜索技术。但是谁会在意它是如何工作的呢？

与其竞争对手不同的是 Vista 的搜索是真正与系统整合的。例如，在 Mac OS X 中，Spotlight 只是在几个关键地方可以获得，如 Mail 和系统配置，在其他诸如帮助里则没有。在 Vista 中，搜索是普遍深入的。当你打开开始菜单并开始输入，搜索将会找到应用程序、文档和其他文件，无需单击屏幕其他任何地方（附图）。的确，你甚至不需要触摸鼠标这也能发生。

如果你想要搜索系统中的文件，单击开始中的搜索即可。它会加载新的搜索窗口，像在开始菜单中一样，你只需要输入要搜索的内容即可。Vista 的搜索是即时返回搜索结果的。高级搜索按钮会下拉一个直观的查询构造器，你可以制定许多选项缩小搜索结果（附图）。

如果你访问遍及系统的任何资源管理器窗口中的搜索框，Vista 的搜索就会在此位置开始。就是说，你可以打开“文档”，就在你的文档集合中进行搜索（附图）或者其他。

想要搜索帮助吗？控制面板呢？实际上，你可以输入诸如"uninstall program"（卸载程序）而找到执行该操作的地方。搜索网络呢？IE也有整合的搜索功能。找到一个网络位置？它是整合的搜索，不是肤浅的整合搜索，它是系统的一部分。

哦，有个更加重要的事情，虽然我必须给微软一个黑色的标记而不是很明显的：你也可以保存搜索结果。这些被保存的搜索作为一个特殊的搜索文件夹出现（默认），但是你可以将其放到任何你想要的地方，包括桌面（附图）。要保存搜索文件夹，你必须访问资源管理器窗口的文件菜单，这里默认是隐藏的（只要敲击ALT键就能看到）。在打开一个搜索文件夹时，它会像普通文件夹一样，虽然它们有着很大的差异。搜索文件夹是动态的，所所以文件和文件夹将会随着文件系统的变化而自动添加或删除。

3. 媒体中心

自从 2002 年一月，Paul 看到首个自由式的演示之后就一直是媒体中心的拥护者，并且从 2002 年 4 月的第一个beta 起他就一直在使用媒体中心。从那时起，Paul 的孩子们就成长在把媒体中心作为电视的世界中。媒体中心提供了你所有记忆——照片、家庭录像和音乐——将其在起居室最大的 TV 上集中展示。

自从 Vista 中新的媒体中心版本出现以来，我就一直迟迟未动。因为早期的问题——我一度将其描述为“丑陋、拥挤和难以辨认”，并且断定微软正在谋杀媒体中心，让它过于复杂和丑陋。从那时起，微软一直在改进 Vista 版本的媒体中心，使其更具魅力和可用。

媒体中心的 Vista 夹在 Windows XP Media Center Edition 2005（Update Rollup 2）和紧随 Windows Vista 之后的版本之间，Paul 的消息是：几乎可以肯定新版媒体中心在2007年假期发布。届时，估计Windows Vista SP1也该发布了。

Paul 为什么会将这个半生不熟的功能作为 Windows Vista RC1 的十大特性之一呢？他的猜测是，一旦新版出现就会是令人兴奋的，尤其是将它连接到大尺寸的 HDTV 电视上并使用它记录 TV（需要电视调谐器）。这绝对是通向未来的一扇窗口，其中的几个区域既是创新的也是引领市场的。苹果一直不断说的数字媒体，微软已经在做了，并且是在起居室里，而且决不低于此。媒体中心是优秀的。

回到“半生不熟”的观点：从使用角度看，媒体中心只是向下一代移植，更多的是可升级和修改的媒体中心UI。媒体中心开始屏幕（附图）和顶级的体验屏幕（录制TV、音乐库和图片库等等）都有了极大的更新，多方向的 UI 很大程度上基于便携媒体中心设备的首次引入。但是，更深层次的 UI 部分仍然停留在 XP MCE 2005 上，并没有新的 UI 更新。例如，设置、视频细节（附图）、图片细节和图片润色等。微软告诉我说整个 UI 都将在下一个媒体中心发布时匹配到新的风格。

不要在转换过程中拖延，媒体中心仍然是完全无可匹敌的。它是 Vista 中最好的功能之一，这一点毫无疑问。

2. Windows Photo Gallery

Windows Photo Gallery（附图）很明显是基于微软的数字图形套件产品。它包括一个管理和排列数字照片和其他图片的逻辑排序，带元数据标签系统，一般它们可以更加有效被管理，也是很好的编辑工具。

在印象中，Windows Photo Gallery 有三个不同的应用程序或模式。这个程序实际上是一个库程序，它可以管理照片集，添加标签和分级。第二个程序是查看器，如果你打开或编辑多数来自 Windows Shell 的图片，就会看到一个最小版本显示单个图片（附图）的图库。这个程序与 XP 的图像预览器很像，除它也可以跳转到 Windows Photo Gallery 的第三种模式：编辑器（附图）外。

在第三种模式中，可以通过单击修复按钮访问，你可以执行单击自动调整或手工编辑曝光度、色彩、裁剪和红眼修复。尤其要提到的是红眼修复功能，它非常优秀，甚至可以移除在 PhotoShop Elements 4 和 Microsoft Digital Image Suite 都会失败的情况下的红眼。不确定为什么 Photo Gallery 如此好，因为它在 Paul 的测试中的确表现得很好。

Windows Photo Gallery 也包括一些其他有用的功能。你可以打印照片、通过邮件发送给他人，也可以将照片烧录到CD 或 DVD，或者制作成照片幻灯影片。你也可以使用其他来自 Windows Photo Gallery 内部的其他程序，所以，如果你选择使用 Windows Photo Gallery 管理你的照片集，可以选择 PhotoShop 等编辑工具，这是你的选择。

苹果迷们将会指出 Windows Photo Gallery 与 iPhoto 十分相似，他们似乎有个观点，事实是微软的 Digital Image Suite 早于 iPhoto 几年时间。不错，iPhoto 提供一些相当简单的功能，而 Windows Photo Gallery 没有，诸如创建漂亮的照片簿。但是，有件事我是同意的，那就是 iPhoto 不是免费的，它需要你购买79美元的iLife套件或者一个新的Mac。

总之，Windows Photo Gallery 相对 XP 的照片功能有了巨大的改进。对于许多人来说，它将是唯一需要的照片工具。

1. Setup 安装

Windows Vista 最早的承诺是可以从 DVD 在约 20 分钟内安装完毕。我们还没有达到这种程度，但是 Vista RC1 可以在不到 20 分钟内使用交互式安装，这要依赖于你的硬件。当然，其中一些属于骗局。Windows Vista 的交互式安装略过了你执行 XP 安装时的一般步骤。不幸的是，你需要在安装完成后执行一些步骤：选取工作组或域、设置其他用户等等。这就是节省时间的大部分内容。

也就是说，许多企业将会简化自动化的 Vista 安装，Vista 基于新的镜像的企业部署方式比 XP 中使用的工具更加好了。最终，安装会因为更加简化而赢得每个人，它是更好的技术，更加快速，总体上，将完成后的配置很正确运行系统留给你。对于像 Paul 一样的人们，因为工作需要有规律的重新安装 Windows，Vista 的安装是个意外，除非你喜欢一个小时或更多停机时间。

总结

Windows Vista 不是完美的。的确，在随后的展示中，Paul 将会重点对这个操作系统的几个最糟糕的特性进行阐述。但是，Windows Vista 是超越 Windows XP 的重大升级和改进。爱或不爱，在未来的几个月或几年中你都将会运行这个新的操作系统。希望这个简短的列表能为你提供给个这么做的理由，或者，至少希望这不是个坏消息。迁移到新的Windows 经常是有所创伤的，那不意味着它不值得。
