101、Install 465 open source fonts（一次安装465种英文开源自体）click System ! Administration ! Software Sources, then the Third-Party Software tab, and click the Add button. Then type the following into the dialog box that appears:
（依次打开“系统-系统管理-软件源”，在“第三方软件”标签中点“添加”，在APT行中输入：）

deb http://ppa.launchpad.net/corenominal/ubuntu hardy main
Click the Add Sources button, then the Close button, and, when prompted, agree to reload the package lists. Then use Synaptic to search for and install the ttf-aenigma package. Once installed the fonts will be available for use straight away in all applications.
（然后点“添加源”，关闭。软件重新更新软件源列表。
用新立得搜索并安装ttf-aenigma，安装完成后系统会自动配置并激活字体。）
