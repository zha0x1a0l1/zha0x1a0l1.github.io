16、Add RAR file compression support to Ubuntu（安装RAR）
只需通过下列命令：
Sudo apt-get install rar unrar
Roller—Ubuntu’s default archive manager—will be able to extract files from RAR archives. You can also use the command from the prompt by simply typing unrar e filename.rar,
（安装好后，系统将支持rar文件的解压缩，也可以通过unrar e filename.rar 命令操作。）
