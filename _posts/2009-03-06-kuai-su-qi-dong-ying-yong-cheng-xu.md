33、Quickly run applications without opening a terminal window（快速启动应用程序）
1、If you want to run a GUI application that doesn’t have a menu entry (for example, gconf-editor), there’s no need to fuss with a terminal window. Just hit Alt + F2 . Then type the name of the program. If it needs to run with root privileges, just type gksu beforehand. If the program is command-line, check the Run In Terminal box.
(快速打开应用程序（例如：运行gconf-editor,firefox,pidgin等）,按快捷键Alt+F2 ,然后输入应用程序的名字（如果应用程序需要管理员权限运行，就在应用程序名称前加上gksu）,回车打开应用程序。如果运行的是命令行,选择窗口右边的“在终端运行”。)
2、Gnome-do
通过新立得搜索并安装gnome-do(最新版本是0.8版，自带了一个漂亮的dock，安装前需要添加APP源)，重起X，按Ctrl+Blackspace键呼出，然后输入所要运行的应用程序的名称回车就可（一般只需输入应用程序名字的前几个字母就可以，输入的同时会有相应的快捷方式图标出现。），会车后，应用程序打开，gnome-do自动关闭。
