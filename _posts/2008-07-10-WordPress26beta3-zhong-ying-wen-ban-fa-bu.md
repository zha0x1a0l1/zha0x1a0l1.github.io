今天，WordPress 开发团队发布了 WordPress 2.6 beta3 版本，这也是公开发布的 WordPress 2.6 的第三个测试版。在此之前，WordPress 将 2.6 正式版的发布日期向后延迟了一周，正式版本将于 7 月14日左右发布。根据 WordPress 2.6 beta3 版本，我们 WordPress中文站 制作了相应的中文版本，供对新版感兴趣的中文用户抢鲜试用。

WordPress 2.6 beta3 版本相对于之前的 beta2 版本主要做了如下更新：
# Image captions（图像标题？我还没看明白 Caption 与 Title 的区别 ）
# Yet more inline documentation（更多内置文档）
# “Add Media” buttons for the RTE full-screen mode （“添加媒体”按钮实时支持全屏模式）
# Theme Preview fixes （修正主题预览功能）
# Gallery sorting fixes （修正相册排序）
# I18N fixes （修正本地化代码）

WordPress 2.6 beta3 英文版下载地址：

http://wordpress.org/wordpress-2.6-beta3.zip

http://wordpress.org/wordpress-2.6-beta3.tar.gz

由于此版本仍然是非正式版本，因此我们不建议用户试用此版本进行升级你的重要博客，仅推荐用于测试环境。

同时，我们依然给网友们提供网上测试博客。测试地址为：http://wp26.wpchina.org/  ，用户名 admin，密码 demo。

测试博客后台的登陆地址为：http://wp26.wpchina.org/wordpress/wp-admin/ 。

在 WordPress 正式发布 2.6 beta3 之前，我们WordPress中文站还依据 WordPress 代码库里的版本，发布过一个 beta3 的测试版，主要用于检测 WordPress 2.6 版本的中文翻译。现在我们再次更新此版本，同时发布单独的中文语言包，仍然希望大家不断对中文版的翻译提出修改意见，来共同打造一个优雅的 WordPress 中文版。
