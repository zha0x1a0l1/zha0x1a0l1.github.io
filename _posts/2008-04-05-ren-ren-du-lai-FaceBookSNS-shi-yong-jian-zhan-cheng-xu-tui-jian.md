1、Openpne：

点评：我推崇的一个开源程序，大约06年开始接触并使用，运行等各方面都很稳定，用户量在十万级别以下均可放心使用。API也非常开放。最强大之处则在于手机支持非常友好。缺点则在于是日本公司开发，汉化目前不是很友好。同时，代码执行效率偏低，结构布局多使用表格而非div+css。一般情况之下，建议有自己的服务器。另外，国内已有商业公司纂改了openpne的开源协议，当做商业软件在销售，建议大家bs此种开源行为。

官方下载网站：http://sourceforge.net/projects/openpne/

官方网站：http://www.openpne.jp/

开发平台：php+mysql

费用：开源+免费

备注：openpne已经成为日本主流的SNS建站程序，有大量的网站都使用它，大家可以参考这个网站，罗列了日本主流的使用openpne的网站：http://www.snsnavi.jp，很多网站的注册用户都过十万了。也有专著讲openpne.

2、OpenKaku：

点评：曾经在Openpne日本公司工作过的中国人郭峰领衔开发的，基于Openpne开发，修正了一些openpne的汉化问题，同时在根据中国用户意见，增强了一些功能。基本结构等各方面均等同于openpne。郭峰是一位很热心的程序员，去年从日本回来之后，也在上海成立了自己的SNS服务运营公司，大家可以通过Openkaku联系他。

官方下载网站：http://www.openkaku.cn/� （注册之后即可下载）

官方网站：http://www.openkaku.cn/

开发平台：php+mysql

费用：免费

备注：郭峰本身也属于SNS的受益者，他运营的华日俱乐部的各项指标均不错。可以算是Openpne国内的成功案例：http://kanichi.jp/

3、UCenter Home

点评：新出的，没有正式运营使用过，不过相信Comsenz Inc.公司的实力。一个未来，整合了ECShop、SupeV、Discuz!的系统，将很好很强大。国内众多使用Discuz!的站长们也可以非常平滑地增加自己的网站的新功能。同时，个人相信，Comsenz Inc.将保持它的一贯竞争力，在SEO、页面静态化处理、整体运行效率以及国内插件整合等各方面远远强于Openpne。

但是，成也萧何，败也萧何。Comsenz Inc.的上述优势也将是UCenter Home很大的缺陷。毕竟，SNS平台与BBS平台有本质差异。（可以参考我的blog：SNS网站与BBS的本质差异）UCenter Home面临的挑战更多来自于：1）站长推广自己的网站与用户信任之间的平衡；2）Comsenz Inc.自身强大的产品线与其他第三方插件或者应用之间的竞争。

对于UCenter Home来说，难点不在于技术，更多在于运营以及架构思路。如能很好解决这些问题，UCenter Home将快速攻占中小网站的SNS市场。

官方网站：http://u.discuz.net/

开发平台：php+mysql

费用：免费

4、Community Server

点评：很不适合创业者或者中小网站使用的一套程序，运营Community Server的三年多时间，让我们吃够了苦头。Community Server架构庞大、复杂、运行效率低，如果只使用一台独立服务器，基本上让用户难以忍受。但是，为什么要推荐Community Server呢？让我们看看Community Server的用户吧：微软、myspace…

Community Server可以说，在企业级的应用这块，它的优势太明显了。Community Server的前身之一，.Text在国内使用太普及了，包括donews的blog系统、cnblogs的blog系统。历经多年考验，也验证了 Community Server的优势。随着Community Server 2008的放出，Community Server在SNS功能上已经达到了一定高度。新版的Community Server 2008将原来的图片库功能与文件库功能进一步整合为一个单独的媒体中心，同时大幅度增强了“我的好友”功能。

官方网站：

开发平台：asp.net2.0+sql server

费用：Express版免费

备注：Community Server的国内成功案例不是很多，目前看到的最成功的应该是金蝶的几个社区。网址如下：http://community.kingdee.com/ 、http://dev.kingdee.com/、http://mfg.kingdee.com/、http://club.youshang.com/

5、CHSNS

点评：SNS程序一般基于php+mysql平台，基于微软的平台很少。除了Community Server之外，另一个可以拿来使用的建站程序是：CHSNS。CHSNS结构很简洁，已有一两个网站已经在使用。但是，如果不尽快开源，基本上是死路一条。

官方网站：http://www.eice.com.cn/

费用：免费下载
